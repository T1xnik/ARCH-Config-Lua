# 1. Конспект настройки

## 1.1 Автогенерация конфига

### Проблема

В начале файла было предупреждение:

```lua
autogenerated = true
```

### Решение

Удалить строку:

```lua
hl.config({
    autogenerated = true
})
```

---

## 1.2 Ошибка в секции input

### Ошибка

```text
unknown config key 'input.kd_layout'
```

### Причина

Опечатка:

```lua
kd_layout
```

вместо

```lua
kb_layout
```

### Исправление

```lua
hl.config({
    input = {
        kb_layout = "us,ru",
        kb_options = "grp:alt_shift_toggle",
        follow_mouse = 1,
    }
})
```

---

## 1.3 Принудительное закрытие окна

### Ошибка

```text
attempt to call a nil value (field 'forcekill')
```

### Причина

Метод отсутствует в текущей версии Hyprland.

### Решение

```lua
hl.bind("SUPER + SHIFT + Q", function()
    local active = hl.get_active_window()

    if active and active.pid then
        hl.exec_cmd("kill -9 " .. active.pid)
    end
end)
```

---

## 1.4 Плавающие окна

### Ошибка

```text
attempt to call a nil value (field 'toggle_floating')
```

### Решение

```lua
hl.bind(
    "SUPER + V",
    hl.dsp.exec_cmd("hyprctl dispatch togglefloating")
)
```

---

## 1.5 Неподдерживаемые поля в window_rule

### Ошибки

```text
unknown match property 'floating'
unknown field 'opacity'
unknown field 'blur'
```

### Причина

Поля не поддерживаются Lua API вашей версии.

### Рабочие правила

```lua
hl.window_rule({
    name = "suppress-maximize-events",
    match = { class = ".*" },
    suppress_event = "maximize",
})

hl.window_rule({
    name = "fix-xwayland-drags",
    match = {
        class = "^$",
        title = "^$",
        xwayland = true,
        float = true
    },
    no_focus = true,
})

hl.window_rule({
    name = "smaller-firefox",
    match = { class = "firefox" },
    size = "1200 800",
    center = true,
})
```

## 1.6 Ошибка resize()

### Ошибка

```text
'x' and 'y' are required
```

### Причина

Вызов без аргументов.

### Решение для мыши

```lua
hl.bind(
    mainMod .. " + mouse:273",
    hl.dsp.window.resize({
        x = 0,
        y = 0
    }),
    { mouse = true }
)
```

### Решение для клавиатуры

```lua
hl.bind(
    mainMod .. " + SHIFT + left",
    hl.dsp.window.resize({
        x = -50,
        y = 0
    })
)

hl.bind(
    mainMod .. " + SHIFT + right",
    hl.dsp.window.resize({
        x = 50,
        y = 0
    })
)
```

# 3. Список удалённых неподдерживаемых полей

| Удалённый код                           | Причина                |
| --------------------------------------- | ---------------------- |
| `match = { floating = true }`           | Свойство отсутствует   |
| `opacity = 0.9`                         | Поле не поддерживается |
| `blur = true`                           | Поле не поддерживается |
| `hl.dsp.window.forcekill()`             | Метод отсутствует      |
| `hl.dsp.window.toggle_floating()`       | Метод отсутствует      |
| `hl.dsp.window.resize()` без параметров | Требуются `x` и `y`    |

# 3. Кнопка смены пользователя

Используется внешний скрипт:

```text
~/.config/hypr/lock_button.sh
```

Содержимое:

```bash
#!/bin/bash

killall hyprlock

sleep 0.3

sudo systemctl restart sddm
```

---

## Сделать исполняемым

```bash
chmod +x ~/.config/hypr/lock_button.sh
```

---

## Отключение запроса пароля

Открыть:

```bash
sudo visudo
```

Добавить:

```text
username ALL=(ALL) NOPASSWD: /usr/bin/systemctl restart sddm
```

Заменить:

```text
username
```

на своё имя пользователя.

# 4. Автоматическая блокировка (Hypridle)

## Установка

```bash
sudo pacman -S hypridle
```

---

## Конфиг

Файл:

```text
~/.config/hypr/hypridle.conf
```

Пример логики:

| Время бездействия | Действие        |
| ----------------- | --------------- |
| 300 сек           | Запуск hyprlock |
| 330 сек           | DPMS Off        |
| 1800 сек          | Suspend         |

---

Пример:

```ini
listener {
    timeout = 300
    on-timeout = hyprlock
}

listener {
    timeout = 330
    on-timeout = hyprctl dispatch dpms off
}

listener {
    timeout = 1800
    on-timeout = systemctl suspend
}
```

# 5. Отключение автовхода SDDM

Если используется автоматический вход в систему, смена пользователя может работать некорректно.

Открыть:

```text
/etc/sddm.conf
```

Найти:

```ini
[Autologin]
```

и закомментировать либо удалить весь блок.

После этого SDDM будет показывать экран входа после выхода из текущей сессии.

# ❌ Типичные ошибки

| Ошибка                                | Причина                              | Решение                                      |
| ------------------------------------- | ------------------------------------ | -------------------------------------------- |
| unknown match property 'floating'     | Поле отсутствует в API               | Удалить правило                              |
| unknown field 'blur'                  | blur не поддерживается в window_rule | Настраивать только через decoration          |
| field 'forcekill'                     | Метод отсутствует                    | Использовать kill -9                         |
| field 'toggle_floating'               | Метод отсутствует                    | Использовать hyprctl dispatch togglefloating |
| input.kd_layout                       | Опечатка                             | Использовать kb_layout                       |
| Unknown keysym                        | Неверный синтаксис бинда             | Использовать + вместо запятых                |
| Обои не отображаются                  | Неверный путь                        | Проверить path                               |
| Нет прозрачности                      | Не настроен blur                     | Включить blur и opacity                      |
| SUPER + L не работает                 | Не установлен hyprlock               | Установить пакет                             |
| Кнопка смены пользователя не работает | Ошибка в sudo или скрипте            | Проверить lock_button.sh                     |
| Не сохраняются скриншоты              | Нет папки назначения                 | Создать каталог                              |
| Конфликт обоев                        | force_default_wallpaper              | Удалить параметр                             |
