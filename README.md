# 1. Конспект настройки

## 1.1 Автогенерация конфига

### Проблема

В начале файла было предупреждение:

```lua
autogenerated = true
```

### Решение

Удалить строку:

```lua
hl.config({
    autogenerated = true
})
```

---

## 1.2 Ошибка в секции input

### Ошибка

```text
unknown config key 'input.kd_layout'
```

### Причина

Опечатка:

```lua
kd_layout
```

вместо

```lua
kb_layout
```

### Исправление

```lua
hl.config({
    input = {
        kb_layout = "us,ru",
        kb_options = "grp:alt_shift_toggle",
        follow_mouse = 1,
    }
})
```

---

## 1.3 Принудительное закрытие окна

### Ошибка

```text
attempt to call a nil value (field 'forcekill')
```

### Причина

Метод отсутствует в текущей версии Hyprland.

### Решение

```lua
hl.bind("SUPER + SHIFT + Q", function()
    local active = hl.get_active_window()

    if active and active.pid then
        hl.exec_cmd("kill -9 " .. active.pid)
    end
end)
```

---

## 1.4 Плавающие окна

### Ошибка

```text
attempt to call a nil value (field 'toggle_floating')
```

### Решение

```lua
hl.bind(
    "SUPER + V",
    hl.dsp.exec_cmd("hyprctl dispatch togglefloating")
)
```

## 1.5 Неподдерживаемые поля в window_rule

### Ошибки

```text
unknown match property 'floating'
unknown field 'opacity'
unknown field 'blur'
```

### Причина

Поля не поддерживаются Lua API вашей версии.

### Рабочие правила

```lua
hl.window_rule({
    name = "suppress-maximize-events",
    match = { class = ".*" },
    suppress_event = "maximize",
})

hl.window_rule({
    name = "fix-xwayland-drags",
    match = {
        class = "^$",
        title = "^$",
        xwayland = true,
        float = true
    },
    no_focus = true,
})

hl.window_rule({
    name = "smaller-firefox",
    match = { class = "firefox" },
    size = "1200 800",
    center = true,
})
```

## 1.6 Ошибка resize()

### Ошибка

```text
'x' and 'y' are required
```

### Причина

Вызов без аргументов.

### Решение для мыши

```lua
hl.bind(
    mainMod .. " + mouse:273",
    hl.dsp.window.resize({
        x = 0,
        y = 0
    }),
    { mouse = true }
)
```

### Решение для клавиатуры

```lua
hl.bind(
    mainMod .. " + SHIFT + left",
    hl.dsp.window.resize({
        x = -50,
        y = 0
    })
)

hl.bind(
    mainMod .. " + SHIFT + right",
    hl.dsp.window.resize({
        x = 50,
        y = 0
    })
)
```

## 1.7 Размер окон и интерфейса

### Проблема

При

```lua
scale = 0.85
```

интерфейс стал слишком мелким.

### Решение

Вернуть:

```lua
scale = 1
```

и увеличить отступы:

```lua
general = {
    gaps_in = 15,
    gaps_out = 30,
}
```

## 1.9 Прозрачность

### Глобальная

```lua
active_opacity = 1.0,
inactive_opacity = 1.0,
```

Можно уменьшить до:

```lua
0.95
```

или

```lua
0.85
```

### Kitty

Файл:

```text
~/.config/kitty/kitty.conf
```

Добавить:

```text
background_opacity 0.85
dynamic_background_opacity yes
```

# 3. Список удалённых неподдерживаемых полей

| Удалённый код                           | Причина                |
| --------------------------------------- | ---------------------- |
| `match = { floating = true }`           | Свойство отсутствует   |
| `opacity = 0.9`                         | Поле не поддерживается |
| `blur = true`                           | Поле не поддерживается |
| `hl.dsp.window.forcekill()`             | Метод отсутствует      |
| `hl.dsp.window.toggle_floating()`       | Метод отсутствует      |
| `hl.dsp.window.resize()` без параметров | Требуются `x` и `y`    |
